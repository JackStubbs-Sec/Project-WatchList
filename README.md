# WatchList

WatchList is a premium personal watch journal built as a mobile-first installable PWA.

## Stack

- React + TypeScript + Vite
- Zustand for app state
- Dexie (IndexedDB) for offline local data
- vite-plugin-pwa for installability and caching

## Quick start

1. Install Node.js 20+.
2. Install dependencies:

```bash
npm install
```

3. Run development server:

```bash
npm run dev
```

4. Build production bundle:

```bash
npm run build
```

## V1 Areas

- Home
- Library
- Add
- Search
- Settings

## Deployment

Use GitHub Pages with the included workflow under `.github/workflows/deploy.yml`.
